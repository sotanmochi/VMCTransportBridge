# Console app examples for VMCTransportBridge

## Build and Run
```
> cd GrpcTransportServer
> dotnet run --configuration Release
```
```
> cd GrpcTransportClient
> dotnet run --configuration Release
```
```
> cd VMCMessageReceiver
> dotnet run --configuration Release --port <port number>
```
```
> cd VMCMessageSender
> dotnet run --configuration Release --port <port number>
```
